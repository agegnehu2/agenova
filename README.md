# AgeNova

AgeNova is a next-generation programming language designed for AI, web development, and modern software engineering.

## ## Features
* 📝 **Simple & Clean Syntax** – Easy to read and write.
* 🚀 **Fast Learning Curve** – Designed to get things done quickly.
* 💻 **Browser Interpreter** – Runs directly in your browser UI without any setup.
* 🔄 **Control Flow Structures** – Supports powerful loops and conditional checks.

---

## ## Current Version: AgeNova v0.0.4 (Latest)

### ### 🚀 What's New in v0.0.4:
* **Conditional Logic (`if` statements):** Execute blocks of code dynamically based on boolean states.
* **Looping Control (`repeat` statements):** Easily loop operations using numbers or stored variables.
* **State Management (`let` variables):** Assign and retrieve dynamic memory seamlessly.

## ⚙️ Installation & Usage

To run AgeNova locally on your machine, follow these steps:

1. Clone this repository:
git clone [https://github.com/agegnehu2/agenova.git](https://github.com/agegnehu2/agenova.git)

2. Navigate into the project directory:
cd agenova

3. Open index.html in your web browser to use the interpreter directly.
